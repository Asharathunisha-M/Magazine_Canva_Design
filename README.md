# Magazine Canva Design: "WE Getting Married"

A beautifully crafted wedding magazine and digital announcement designed using Canva. This repository hosts the digital assets for the "WE Getting Married" project, available in both interactive video and high-quality print/digital document formats.

---

## 📂 Repository Contents

*   **`WE Getting married.pdf`**: The high-resolution PDF version of the magazine design, perfect for viewing as a digital flipbook or printing.
*   **`WE Getting married.mp4`**: An animated video version of the magazine showcase, highlighting the transitions, layout, and visual story.

---

## 🎨 About the Design

This project was created using **Canva** to deliver a modern, elegant, and personalized wedding magazine experience. 

*   **Design Tool:** Canva
*   **Theme:** Wedding / Announcement / Elegant Keepsake
*   **Formats:** PDF (Document) & MP4 (Video Presentation)

---

## 🚀 How to View

1. **PDF Version:** Click on `WE Getting married.pdf` above and click the **Download** button to view or print the full layout.
2. **Video Presentation:** Click on `WE Getting married.mp4` to watch the animated transition preview directly inside your browser.
